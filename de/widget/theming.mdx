Das `<ribaunt-widget>` verwendet Shadow DOM zur Stilkapselung, was bedeutet, dass externe Stylesheets nicht versehentlich in die Interna des Widgets durchsickern können. Sie können sein gesamtes Erscheinungsbild dennoch anpassen, indem Sie CSS-Custom-Properties (Variablen) am `ribaunt-widget`-Host-Element setzen — Shadow DOM erbt diese von außen, sodass Ihre Überschreibungen sauber durchfließen.

## Schnellstart

Die folgenden vier Variablen reichen für ein minimales Dark-Theme aus:

```css
ribaunt-widget {
  --ribaunt-background: #1e1e1e;
  --ribaunt-border-color: #444;
  --ribaunt-color: #f0f0f0;
  --ribaunt-spinner-color: #f0f0f0;
}
```

| `--ribaunt-widget-height` | `58px` | Widget-Höhe |
| `--ribaunt-widget-padding` | `14px` | Inneres Padding des Widgets |
| `--ribaunt-gap` | `15px` | Abstand zwischen Checkbox und Text |
| `--ribaunt-checkbox-size` | `25px` | Breite und Höhe der Checkbox |

## Dark Mode

Ribaunt erzwingt keinen automatischen Dark Mode. Stattdessen binden Sie ihn selbst ein — entweder über eine Media Query, die auf die OS-Präferenz reagiert, oder über eine CSS-Klasse, die Ihre Anwendung steuert.

### Über eine Media Query

```css
@media (prefers-color-scheme: dark) {
  ribaunt-widget {
    --ribaunt-background: #2d2d2d;
    --ribaunt-border-color: #555;
    --ribaunt-color: #ffffff;
    --ribaunt-checkbox-background: #444;
    --ribaunt-spinner-color: #ffffff;
    --ribaunt-spinner-background-color: #333;
    --ribaunt-logo-color: #cccccc;
  }
}
```

### Über eine CSS-Klasse

Wenn Ihre App einen eigenen Dark-Mode-Schalter hat, der eine Klasse zu einem übergeordneten Element hinzufügt (z. B. `<html class="dark">`), können Sie die Überschreibungen auf diese Klasse beschränken:

```css
.dark ribaunt-widget {
  --ribaunt-background: #111;
  --ribaunt-color: #eee;
}
```

## Benutzerdefinierte Dimensionen

Sie können das Widget so anpassen, dass es in Ihr Formularlayout passt, indem Sie die Dimensionsvariablen überschreiben. Das folgende Beispiel erstellt eine `.large`-Variante:

```css
ribaunt-widget.large {
  --ribaunt-widget-width: 300px;
  --ribaunt-widget-height: 72px;
  --ribaunt-checkbox-size: 32px;
}
```

<Tip>
  Standard-CSS-Eigenschaften wie `margin`, `box-shadow` und `display` funktionieren direkt am `ribaunt-widget`-Host-Element, da es sich um ein Inline-Block-Custom-Element handelt. Sie benötigen kein umschließendes `<div>`, nur um Abstände oder Schatten anzuwenden.
</Tip>
