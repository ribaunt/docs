Ribaunt ist eine zustandslose Proof-of-Work-CAPTCHA-Bibliothek für Node.js und moderne Browser. Dein Server stellt Challenges als signierte JWTs aus, das Browser-Widget löst das PoW-Rätsel, und dein Server verifiziert den eingereichten Nachweis – ganz ohne Datenbank. Setze es vor jedes Formular, jeden Anmeldevorgang oder API-Endpunkt, um Bots zu stoppen und gleichzeitig echten Nutzern ein reibungsloses Erlebnis zu bieten.

## In vier Schritten startklar

<Steps>
  <Step title="Ribaunt installieren">
    Füge das Paket mit deinem bevorzugten Paketmanager zu deinem Projekt hinzu.

    ```bash
    npm install ribaunt
    ```
  </Step>
  <Step title="RIBAUNT_SECRET festlegen">
    Hinterlege ein starkes, zufällig generiertes Secret in deiner Serverumgebung. Ribaunt verwendet es, um JWT-Challenge-Tokens zu signieren und zu verifizieren.

    ```env
    RIBAUNT_SECRET="replace-with-a-long-random-secret"
    ```
  </Step>
  <Step title="Server-Endpunkte hinzufügen">
    Erstelle einen `GET /api/captcha/challenge`-Endpunkt, der Challenges ausstellt, und einen `POST /api/captcha/verify`-Endpunkt, der Lösungen validiert.

    ```ts
    import { createChallenge, verifySolution } from 'ribaunt';

    app.get('/api/captcha/challenge', (_req, res) => {
      const challenges = createChallenge({ difficulty: 5, amount: 4, ttlSeconds: 120 });
      res.json({ challenges });
    });

    app.post('/api/captcha/verify', async (req, res) => {
      const { tokens, solutions } = req.body;
      const result = await verifySolution(tokens, solutions);

      if (!result.valid) {
        return res.status(400).json({ success: false, error: result.reason, message: result.message });
      }

      return res.json({ success: true });
    });
    ```
  </Step>
  <Step title="Widget einbinden">
    Füge die Web-Komponente in deine HTML-Seite ein. Sie lädt eine Challenge, löst sie im Browser und sendet ein `verify`-Event, sobald sie fertig ist.

    ```html
    <script type="module" src="/node_modules/ribaunt/dist/widget-browser.js"></script>

    <ribaunt-widget
      challenge-endpoint="/api/captcha/challenge"
      verify-endpoint="/api/captcha/verify"
      auto-verify="true"
    ></ribaunt-widget>

    <script>
      document.querySelector('ribaunt-widget')
        .addEventListener('verify', (event) => {
          console.log('Verified!', event.detail.solutions);
        });
    </script>
    ```
  </Step>
</Steps>
