Ribaunt ist eine zustandslose Proof-of-Work-CAPTCHA-Bibliothek für Node.js und moderne Browser. Ihr Server stellt Challenges als signierte JWTs aus, das Browser-Widget löst das PoW-Rätsel, und Ihr Server überprüft den eingereichten Nachweis — alles ohne Datenbankzugriff. Setzen Sie es vor jedes Formular, jeden Anmeldevorgang oder jeden API-Endpunkt, um Bots abzuwehren und gleichzeitig echten Nutzern ein reibungsloses Erlebnis zu bieten.

## In vier Schritten einsatzbereit

<Steps>
  <Step title="Ribaunt installieren">
    Fügen Sie das Paket mit Ihrem bevorzugten Paketmanager zu Ihrem Projekt hinzu.

    ```bash
    npm install ribaunt
    ```
  </Step>
  <Step title="RIBAUNT_SECRET festlegen">
    Fügen Sie ein starkes, zufällig generiertes Geheimnis zu Ihrer Server-Umgebung hinzu. Ribaunt verwendet es zum Signieren und Verifizieren von JWT-Challenge-Token.

    ```env
    RIBAUNT_SECRET="replace-with-a-long-random-secret"
    ```
  </Step>
  <Step title="Server-Endpunkte hinzufügen">
    Erstellen Sie einen `GET /api/captcha/challenge`-Endpunkt, der Challenges ausstellt, sowie einen `POST /api/captcha/verify`-Endpunkt, der Lösungen validiert.

    ```ts
    import { createChallenge, verifySolution } from 'ribaunt';

    app.get('/api/captcha/challenge', (_req, res) => {
      const challenges = createChallenge(5, 4, 120);
      res.json({ challenges });
    });

    app.post('/api/captcha/verify', async (req, res) => {
      const { tokens, solutions } = req.body;
      const valid = await verifySolution(tokens, solutions);
      if (!valid) return res.status(400).json({ success: false });
      return res.json({ success: true });
    });
    ```
  </Step>
  <Step title="Widget einbinden">
    Fügen Sie die Web-Komponente in Ihre HTML-Seite ein. Sie ruft eine Challenge ab, löst sie im Browser und sendet ein `verify`-Event, wenn sie fertig ist.

    ```html
    <script type="module" src="/node_modules/ribaunt/dist/widget-browser.js"></script>

    <ribaunt-widget
      challenge-endpoint="/api/captcha/challenge"
      verify-endpoint="/api/captcha/verify"
      auto-verify="true"
    ></ribaunt-widget>

    <script>
      document.querySelector('ribaunt-widget')
        .addEventListener('verify', (e) => {
          console.log('Verified!', e.detail.solutions);
        });
    </script>
    ```
  </Step>
</Steps>
