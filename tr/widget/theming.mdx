`<ribaunt-widget>`, stil kapsüllemesi için Shadow DOM kullanır; bu da harici stil sayfalarının widget'ın iç yapılarına yanlışlıkla sızamayacağı anlamına gelir. Yine de tüm görünümünü `ribaunt-widget` host öğesinde CSS özel özellikleri (değişkenler) ayarlayarak özelleştirebilirsiniz — Shadow DOM bunları dışarıdan miras alır, böylece geçersiz kılmalarınız temiz bir şekilde akar.

## Hızlı başlangıç

Aşağıdaki dört değişken minimal bir karanlık tema için yeterlidir:

```css
ribaunt-widget {
  --ribaunt-background: #1e1e1e;
  --ribaunt-border-color: #444;
  --ribaunt-color: #f0f0f0;
  --ribaunt-spinner-color: #f0f0f0;
}
```

| Değişken | Varsayılan | Açıklama |
|---|---|---|
| `--ribaunt-background` | `#fdfdfd` | Widget konteyner arka planı |

| `--ribaunt-border-radius` | `14px` | Widget kenarlık yarıçapı |

| `--ribaunt-widget-width` | `230px` | Widget genişliği |
| `--ribaunt-widget-height` | `58px` | Widget yüksekliği |
| `--ribaunt-widget-padding` | `14px` | Widget iç dolgusu |
| `--ribaunt-gap` | `15px` | Onay kutusu ve metin arasındaki boşluk |
| `--ribaunt-checkbox-size` | `25px` | Onay kutusu genişliği ve yüksekliği |
| `--ribaunt-checkbox-background` | `#fafafa91` | Onay kutusu arka plan rengi |

## Karanlık mod

Ribaunt otomatik bir karanlık mod dayatmaz. Bunun yerine, bunu kendiniz bağlarsınız — ya işletim sistemi tercihine yanıt veren bir medya sorgusu ile ya da uygulamanızın kontrol ettiği bir CSS sınıfı ile.

### Medya sorgusu kullanma

```css
@media (prefers-color-scheme: dark) {
  ribaunt-widget {
    --ribaunt-background: #2d2d2d;
    --ribaunt-border-color: #555;
    --ribaunt-color: #ffffff;
    --ribaunt-checkbox-background: #444;
    --ribaunt-spinner-color: #ffffff;
    --ribaunt-spinner-background-color: #333;
    --ribaunt-logo-color: #cccccc;
  }
}
```

### CSS sınıfı kullanma

Uygulamanızın bir üst öğeye sınıf ekleyen kendi karanlık mod geçişi varsa (örneğin `<html class="dark">`), geçersiz kılmaları bu sınıfla kapsamlandırabilirsiniz:

```css
.dark ribaunt-widget {
  --ribaunt-background: #111;
  --ribaunt-color: #eee;
}
```

## Özel boyutlar

Widget'ı form düzeninize uyacak şekilde boyut değişkenlerini geçersiz kılarak yeniden boyutlandırabilirsiniz. Aşağıdaki örnek bir `.large` varyantı oluşturur:

```css
ribaunt-widget.large {
  --ribaunt-widget-width: 300px;
  --ribaunt-widget-height: 72px;
  --ribaunt-checkbox-size: 32px;
}
```

<Tip>
`margin`, `box-shadow` ve `display` gibi standart CSS özellikleri, satır içi blok bir özel öğe olduğundan `ribaunt-widget` host öğesinde doğrudan çalışır. Boşluk veya gölgeler uygulamak için sarmalayıcı bir `<div>`'e ihtiyacınız yoktur.
</Tip>
