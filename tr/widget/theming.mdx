`<ribaunt-widget>`, stil kapsüllemesi için Shadow DOM kullanır; bu da harici stil sayfalarının widget'ın iç kısımlarına yanlışlıkla sızamayacağı anlamına gelir. Yine de `ribaunt-widget` ana öğesinde CSS özel özellikleri (değişkenler) ayarlayarak tüm görünümünü özelleştirebilirsiniz — Shadow DOM bunları dışarıdan miras aldığından, geçersiz kılma işlemleriniz temiz bir şekilde akar.

## Hızlı başlangıç

Aşağıdaki dört değişken minimal bir karanlık tema için yeterlidir:

```css
ribaunt-widget {
  --ribaunt-background: #1e1e1e;
  --ribaunt-border-color: #444;
  --ribaunt-color: #f0f0f0;
  --ribaunt-spinner-color: #f0f0f0;
}
```

| `--ribaunt-checkbox-size` | `25px` | Onay kutusu genişliği ve yüksekliği |

## Karanlık mod

Ribaunt, otomatik karanlık mod uygulamaz. Bunun yerine, kendiniz bağlarsınız — ya OS tercihine yanıt veren bir media query aracılığıyla ya da uygulamanızın kontrol ettiği bir CSS sınıfı aracılığıyla.

### Media query kullanma

```css
@media (prefers-color-scheme: dark) {
  ribaunt-widget {
    --ribaunt-background: #2d2d2d;
    --ribaunt-border-color: #555;
    --ribaunt-color: #ffffff;
    --ribaunt-checkbox-background: #444;
    --ribaunt-spinner-color: #ffffff;
    --ribaunt-spinner-background-color: #333;
    --ribaunt-logo-color: #cccccc;
  }
}
```

### CSS sınıfı kullanma

Uygulamanızın bir üst öğeye sınıf ekleyen kendi karanlık mod geçiş düğmesi varsa (örneğin `<html class="dark">`), geçersiz kılma işlemlerini bu sınıfa göre kapsamlandırabilirsiniz:

```css
.dark ribaunt-widget {
  --ribaunt-background: #111;
  --ribaunt-color: #eee;
}
```

## Özel boyutlar

Boyut değişkenlerini geçersiz kılarak widget'ı form düzeninize sığacak şekilde yeniden boyutlandırabilirsiniz. Aşağıdaki örnek bir `.large` varyantı oluşturur:

```css
ribaunt-widget.large {
  --ribaunt-widget-width: 300px;
  --ribaunt-widget-height: 72px;
  --ribaunt-checkbox-size: 32px;
}
```

<Tip>
  `margin`, `box-shadow` ve `display` gibi standart CSS özellikleri, `ribaunt-widget` ana öğesi üzerinde doğrudan çalışır çünkü o bir inline-block özel öğedir. Sadece boşluk veya gölge uygulamak için bir sarmalayıcı `<div>`'e ihtiyacınız yoktur.
</Tip>
