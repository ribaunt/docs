---

`solveChallenge()`, tarayıcı widget'ı tarafından kullanılan proof-of-work algoritmasının aynısını çalıştırır; ancak Node.js'te eşzamanlı olarak. Challenge ve verify endpoint'lerinizin otomatik testi için tasarlanmıştır — istek işleyicilerinde üretim kullanımı için değil.

## İçe aktarma

```ts
import { solveChallenge } from 'ribaunt';
```

## İmza

```ts
// Single token
function solveChallenge(
  token: ChallengeToken,
  options?: SolveChallengeOptions
): ChallengeSolution | undefined

// Array of tokens
function solveChallenge(
  token: ChallengeToken[],
  options?: SolveChallengeOptions
): ChallengeSolution[] | undefined
```

## Parametreler

<ParamField path="token" type="ChallengeToken | ChallengeToken[]" required>
  `createChallenge()`'tan tek bir JWT challenge token'ı veya token'lardan oluşan bir dizi.
</ParamField>

<ParamField path="options" type="SolveChallengeOptions">
  Uzun süreli eşzamanlı çözümleri önlemek için isteğe bağlı korumalar.

  <Expandable title="SolveChallengeOptions">
    <ParamField path="maxIterations" type="number">
      Token başına nonce denemelerinin kesin üst sınırı. Ulaşıldığında `undefined` döndürür.
    </ParamField>

    <ParamField path="maxDurationMs" default="30000" type="number">
      Token başına milisaniye cinsinden maksimum eşzamanlı çözüm süresi. Aşıldığında `undefined` döndürür.
    </ParamField>
  </Expandable>
</ParamField>

## Dönüş değeri

Tek bir token girişi için `ChallengeSolution` (`{ nonce: string; hash: string }`), bir dizi girişi için `ChallengeSolution[]` döndürür. Herhangi bir korumaya ulaşılırsa veya bir token geçersizse `undefined` döndürür. Bir diziyi çözerken, tek bir token başarısız olur olmaz `undefined` döndürülür — kısmi sonuçlar döndürülmez.

## Örnek

```ts
import { createChallenge, solveChallenge, verifySolution } from 'ribaunt';

// Use low difficulty in tests for speed
const tokens = createChallenge(3, 2, 60);
const solutions = solveChallenge(tokens);

if (solutions) {
  const valid = await verifySolution(tokens, solutions);
  console.log('Valid:', valid); // true
}
```

Korumalarla:

```ts
const solution = solveChallenge(token, {
  maxDurationMs: 2000,
  maxIterations: 500_000,
});

if (!solution) {
  console.log('Solver gave up — difficulty too high or timeout reached');
}
```

<Warning>
  `solveChallenge` eşzamanlı ve CPU yoğundur. Onu asla bir üretim HTTP istek işleyicisinde çağırmayın — Node.js event loop'unuzu bloke eder.
</Warning>

<Tip>
  Testlerde zorluk 3–4 kullanın. Zorluk 5, test paketinizi gözle görülür şekilde yavaşlatır.
</Tip>
