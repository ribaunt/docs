Ribaunt, Node.js ve modern tarayıcılar için durumsuz bir iş kanıtı (proof-of-work) CAPTCHA kütüphanesidir. Sunucunuz meydan okumaları imzalı JWT'ler olarak yayınlar, tarayıcı widget'ı PoW bulmacasını çözer ve sunucunuz gönderilen kanıtı doğrular — hepsi bir veritabanına dokunmadan. Botları durdururken gerçek kullanıcılar için akıcı bir deneyim sürdürmek üzere herhangi bir formun, kayıt akışının veya API uç noktasının önüne ekleyin.

## Dört adımda çalışır hale getirin

<Steps>
  <Step title="Ribaunt'u kurun">
    Paketi tercih ettiğiniz paket yöneticisi ile projenize ekleyin.

    ```bash
    npm install ribaunt
    ```
  </Step>
  <Step title="RIBAUNT_SECRET'i ayarlayın">
    Sunucu ortamınıza güçlü, rastgele oluşturulmuş bir gizli anahtar ekleyin. Ribaunt, JWT meydan okuma jetonlarını imzalamak ve doğrulamak için bunu kullanır.

    ```env
    RIBAUNT_SECRET="replace-with-a-long-random-secret"
    ```
  </Step>
  <Step title="Sunucu uç noktaları ekleyin">
    Meydan okumalar yayınlayan bir `GET /api/captcha/challenge` uç noktası ve çözümleri doğrulayan bir `POST /api/captcha/verify` uç noktası oluşturun.

    ```ts
    import { createChallenge, verifySolution } from 'ribaunt';

    app.get('/api/captcha/challenge', (_req, res) => {
      const challenges = createChallenge(5, 4, 120);
      res.json({ challenges });
    });

    app.post('/api/captcha/verify', async (req, res) => {
      const { tokens, solutions } = req.body;
      const valid = await verifySolution(tokens, solutions);
      if (!valid) return res.status(400).json({ success: false });
      return res.json({ success: true });
    });
    ```
  </Step>
  <Step title="Widget'ı ekleyin">
    Web bileşenini HTML sayfanıza ekleyin. Bir meydan okuma getirir, tarayıcıda çözer ve tamamlandığında bir `verify` olayı yayar.

    ```html
    <script type="module" src="/node_modules/ribaunt/dist/widget-browser.js"></script>

    <ribaunt-widget
      challenge-endpoint="/api/captcha/challenge"
      verify-endpoint="/api/captcha/verify"
      auto-verify="true"
    ></ribaunt-widget>

    <script>
      document.querySelector('ribaunt-widget')
        .addEventListener('verify', (e) => {
          console.log('Verified!', e.detail.solutions);
        });
    </script>
    ```
  </Step>
</Steps>
