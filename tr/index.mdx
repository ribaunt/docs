Ribaunt, Node.js ve modern tarayıcılar için durumsuz bir iş kanıtı (proof-of-work) CAPTCHA kütüphanesidir. Sunucunuz challenge'ları imzalı JWT'ler olarak yayınlar, tarayıcı widget'ı PoW bulmacasını çözer ve sunucunuz gönderilen kanıtı doğrular — hepsi bir veritabanına dokunmadan. Herhangi bir formun, kayıt akışının veya API uç noktasının önüne koyarak gerçek kullanıcılar için akıcı bir deneyim sunarken botları engelleyin.

## Dört adımda çalışır hale getirin

<Steps>
  <Step title="Ribaunt'u yükleyin">
    Tercih ettiğiniz paket yöneticisiyle paketi projenize ekleyin.

    ```bash
    npm install ribaunt
    ```
  </Step>
  <Step title="RIBAUNT_SECRET değerini ayarlayın">
    Sunucu ortamınıza güçlü, rastgele oluşturulmuş bir gizli anahtar ekleyin. Ribaunt bunu JWT challenge tokenlerini imzalamak ve doğrulamak için kullanır.

    ```env
    RIBAUNT_SECRET="replace-with-a-long-random-secret"
    ```
  </Step>
  <Step title="Sunucu uç noktaları ekleyin">
    Challenge'ları yayınlayan bir `GET /api/captcha/challenge` uç noktası ve çözümleri doğrulayan bir `POST /api/captcha/verify` uç noktası oluşturun.

    ```ts
    import { createChallenge, verifySolution } from 'ribaunt';

    app.get('/api/captcha/challenge', (_req, res) => {
      const challenges = createChallenge(5, 4, 120);
      res.json({ challenges });
    });

    app.post('/api/captcha/verify', async (req, res) => {
      const { tokens, solutions } = req.body;
      const valid = await verifySolution(tokens, solutions);
      if (!valid) return res.status(400).json({ success: false });
      return res.json({ success: true });
    });
    ```
  </Step>
  <Step title="Widget'ı yerleştirin">
    HTML sayfanıza web bileşenini ekleyin. Bir challenge çeker, tarayıcıda çözer ve tamamlandığında bir `verify` olayı yayar.

    ```html
    <script type="module" src="/node_modules/ribaunt/dist/widget-browser.js"></script>

    <ribaunt-widget
      challenge-endpoint="/api/captcha/challenge"
      verify-endpoint="/api/captcha/verify"
      auto-verify="true"
    ></ribaunt-widget>

    <script>
      document.querySelector('ribaunt-widget')
        .addEventListener('verify', (e) => {
          console.log('Verified!', e.detail.solutions);
        });
    </script>
    ```
  </Step>
</Steps>
